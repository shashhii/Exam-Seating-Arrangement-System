## About

Automates exam hall seating arrangements for colleges. Developed this after seeing how manually managing seating was a pain for our department.

## Tech Stack

Java, JSP, Servlet, MySQL

## Getting Started

```bash
git clone https://github.com/shashhii/Exam-Seating-Arrangement-System.git
cd Exam-Seating-Arrangement-System
```

- Clean and intuitive UI
- Fully functional backend
- Responsive design
- Well-structured codebase
